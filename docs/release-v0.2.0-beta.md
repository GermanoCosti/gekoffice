# GekOffice v0.2.0-beta

Prima personalizzazione operativa di GekOffice basata su LibreOffice.

## Contenuto

- profilo LibreOffice dedicato a GekOffice
- macro integrate per i principali modelli PA
- estensione `Strumenti PA` per accesso rapido ai template
- template iniziali accessibili con corpo minimo 12 pt
- branding coordinato del progetto
- script di installazione e avvio per Windows

## Modelli inclusi

- determina
- delibera
- verbale
- lettera istituzionale
- relazione tecnica

## Obiettivo

Questa versione introduce la prima base installabile e operativa del progetto, con accesso rapido ai modelli documentali per la Pubblica Amministrazione italiana.

## Stato

Versione beta iniziale per validazione funzionale e rifinitura dell'esperienza utente.
